# 灵气护盾纹理绘制指南

## 文件位置
`src/main/resources/assets/tiandao/textures/spell/hexagon_shield.png`

## 纹理规格

### 基础参数
- **尺寸**: 256x256 像素 或 512x512 像素（推荐512x512以获得更好的细节）
- **格式**: PNG，带透明通道（Alpha Channel）
- **颜色模式**: RGBA（红、绿、蓝、透明度）

## 设计理念

护盾纹理应该设计成**六边形蜂窝网格**效果，类似科幻电影中的能量护盾。由于纹理会映射到球体上，需要考虑：

1. **无缝重复**: 纹理的上下左右边缘应该能够无缝连接
2. **中心对称**: 纹理中心应该有焦点效果
3. **渐变透明**: 边缘应该逐渐透明，中间较不透明

## 绘制步骤

### 第1步：创建基础网格

1. 创建新画布（512x512像素）
2. 填充深蓝色背景（RGB: 20, 40, 80，透明度：200）

### 第2步：绘制六边形网格

#### 方法A：手绘六边形（推荐用于独特风格）

1. **六边形框架**
   - 使用浅蓝色（RGB: 100, 150, 255）绘制六边形边框
   - 线宽：2-3像素
   - 在画布上均匀分布六边形，形成蜂窝状网格
   - 建议：从中心向外绘制，保持对称

2. **网格节点**
   - 在每个六边形顶点绘制发光点
   - 颜色：亮蓝色（RGB: 150, 200, 255）
   - 大小：4-6像素的圆形
   - 添加外发光效果

#### 方法B：程序生成六边形（推荐用于精确网格）

使用图像编辑软件的"图案填充"功能或脚本生成六边形网格：

```
六边形参数：
- 边长：约30-40像素
- 线宽：2像素
- 颜色：RGB(100, 150, 255)
- 间距：均匀分布
```

### 第3步：添加能量效果

1. **中心光晕**
   - 在纹理中心（256, 256）添加圆形光晕
   - 使用径向渐变：中心白色 → 外围蓝色 → 透明
   - 半径：约50-80像素

2. **能量流动线条**
   - 沿着六边形网格添加流动的能量线条
   - 颜色：亮青色（RGB: 100, 255, 255）
   - 使用柔和的画笔，添加运动模糊效果

3. **边缘渐变**
   - 纹理边缘应该逐渐变透明
   - 使用"渐变蒙版"：中心100%不透明 → 边缘0%不透明
   - 边缘渐变区域：约30-50像素

### 第4步：颜色调整

**主色调**（六边形网格）：
- 基础蓝色：RGB(100, 150, 255)
- 发光蓝色：RGB(150, 200, 255)
- 高光白色：RGB(200, 220, 255)

**辅助色调**（能量效果）：
- 青色：RGB(100, 255, 255)
- 紫色：RGB(150, 100, 255)（可选，用于特殊效果）

### 第5步：透明通道设置

1. **六边形网格区域**
   - 不透明度：60-80%（让护盾半透明，不遮挡视线）

2. **背景区域**
   - 不透明度：20-40%（几乎透明，只保留淡淡的蓝色调）

3. **边缘区域**
   - 不透明度：0%（完全透明，实现平滑过渡）

### 第6步：测试和调整

1. 在游戏中测试纹理效果
2. 检查：
   - 纹理是否无缝重复
   - 透明度是否合适（不应该遮挡玩家视野）
   - 颜色是否在游戏中看起来自然
   - 六边形网格是否清晰可见

## 常见问题

**Q: 纹理在球体上看起来很扭曲？**
A: 这是正常的，球体UV映射会在极点产生扭曲。可以：
- 在极点区域使用更简单的图案
- 或者接受这种扭曲作为风格特点

**Q: 护盾太透明/太不透明？**
A: 调整整个图层的透明度：
- 太透明：增加RGB值，减少Alpha值
- 太不透明：减少RGB值，增加Alpha值

**Q: 纹理边缘有白边？**
A: 确保边缘像素的Alpha值为0，或者使用"抗锯齿"功能平滑边缘。
